Card - Best Practices

Headers:

  • If you don't have an avatar don't use <mat-card-header>.
  • If you have a subtitle and no image use <mat-card-title-group>.
  • If you only have a <mat-card-title> then it doesn't need to be nested in a parent header/group.

Content

  • Always use <mat-card-content>.
